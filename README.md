## Simple HTML Website

This project is basic multi-stage website built using **HTML only**.
The goal is to learn how to structure a website semantically before adding any styling.

The project description can be found in roadmap.sh -> https://roadmap.sh/projects/basic-html-website

---

## Project Structure
/website 
├── index.html        # Homepage 
├── projects.html     # Projects page 
├── articles.html     # Articles page 
└── contact.html      # Contact page

---

## Goals
- Learn how to create multiple pages in a website
- Structure a website using semantic HTML (`header`, `nav`, `main`, `section`, `article`, `footer`).
- Add SEO meta tags to each page.
- Prepare the site for styling later with CSS

---

## Pages
- **Homepage (`index.html`)**
    Contains an introduction and three sections:
    - Projects (left)
    - Work Experience (middle)
    - Education (right)

- **Projects (`projects.html`)**
    Listing sample projects with description

- **Articles (`articles.html`)**
    Contains sample articles or blog posts

---
## How to Run
1. Clone or download this repository
2. Open any of the `.html` files in your browser (e.g, double-click `index.html`).
3. Navigate between pages using the navigation bar

---
